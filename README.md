# ALX Listing App

An Airbnb clone listing page built with Next.js, TypeScript, and Tailwind CSS.

## Project Structure

- `components/common/`: Reusable UI components like Card and Button.
- `interfaces/`: TypeScript interfaces for type safety.
- `constants/`: Application constants and configuration.
- `public/assets/`: Static assets like images and SVGs.
- `pages/`: Next.js pages (Pages Router).
- `styles/`: Global styles including Tailwind CSS imports.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
